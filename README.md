(i) Essential Attributes
Three attributes that were essential for making my website functional were href, src, and alt. The href attribute was important for creating links using the <a> tag, allowing users to navigate to external websites. The src attribute was used in the <img> tag to display images on my webpage. The alt attribute was also important because it provided alternative text in case an image failed to load, improving accessibility.
(ii) How i used class and id element
I used the class attribute to group similar elements together, such as movies belonging to the same genre like horror, comedy, and cartoon. This helped me organize my content more effectively. On the other hand, I used the id attribute for unique elements, such as specific sections of the page, so that they could be easily identified and referenced.
(iii) Attribute that Improved User Experience
The alt attribute improved user experience the most because it ensures that users can still understand what an image represents even if it does not load. It also helps users who rely on screen readers, making the website more accessible.
